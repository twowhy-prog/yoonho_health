# 윤호랑 하루하루 - 매일의 사랑, 매일의 기록

아이의 알레르기와 건강을 체계적으로 관리하는 웹 애플리케이션입니다.

## 🌟 주요 기능

### 📝 일일 기록 관리
- 음식 섭취 기록
- 알레르기 증상 추적
- 심각도 점수 (1-10점)
- 복용 약물 기록
- 사진 첨부 (최대 2장)
- 작성자 구분 (아빠/엄마)

### 💊 투약 알림
- 정시 브라우저 알림
- 복용 완료 체크
- 알림 활성화/비활성화
- 다음 투약 시간 안내

### 📊 월간 통계 분석
- 음식별 위험도 랭킹
- 일별 반응 추이
- 개선 제안사항
- 안전/위험 음식 분류

### 🌙 다크 모드
- 라이트/다크 테마 전환
- 시스템 테마 연동

### 💾 백업 & 복원
- 로컬 백업 (JSON 파일)
- 데이터 복원
- CSV 내보내기

## 🚀 배포 방법

### Vercel 배포 (추천)
1. GitHub에 코드 업로드
2. Vercel.com에서 GitHub 연동
3. 자동 배포 완료

### 로컬 실행
\`\`\`bash
npm install
npm run dev
\`\`\`

## 📱 사용법

1. **작성자 선택**: 헤더에서 아빠/엄마 선택
2. **기록 추가**: 음식, 증상, 약물 등 입력
3. **투약 알림**: 정시 알림 설정
4. **통계 확인**: 월간 패턴 분석
5. **정기 백업**: 데이터 안전 보관

## ⚠️ 주의사항

- 심각한 알레르기 반응 시 즉시 병원 방문
- 이 앱은 기록 목적이며 의료 상담을 대체하지 않음
- 정기적인 데이터 백업 권장

## 💝 개발 목적

소중한 아이의 건강한 성장을 위한 체계적인 기록 관리 도구
